# NestMusicPlayer
### 仿Spotify UI 音乐在线播放网站 
#### 感谢 Binaryify node版网易云api
https://github.com/Binaryify/NeteaseCloudMusicApi

## 项目依赖安装
```
npm install
```

### 开发环境运行、热重载
```
npm run serve
```

### 生产环境打包
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
